This project presents an interactive analytics dashboard developed using Microsoft Power BI to analyze EV charging station infrastructure, demand patterns, customer experience, and operational efficiency.

рҹҺҜ Project Objective
-----------------------

The objective of this project is to analyze EV charging station data to understand:

1. Charging demand distribution across cities

2. Infrastructure availability and capacity

3. Customer waiting time patterns

4. Peak usage hours

5. Impact of local events on charging demand

6. Influence of nearby amenities on station utilization

The analysis helps identify high-demand locations, infrastructure gaps, and operational stress points in the charging network.
